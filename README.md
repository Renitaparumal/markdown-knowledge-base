# Collaborative Markdown Knowledge Base

## Project Description
This project is a collaborative Markdown knowledge base built using Git and GitHub to demonstrate version control, branching, pull requests, and teamwork workflows.

---

## Project Structure
- chapter1.md (Renita)
- chapter2.md (Dev2)
- chapter3.md (Collaboration)

---

## Git Workflow
- Created branches for each feature (feature-renita, feature-dev2)
- Used Pull Requests for merging changes
- Practiced code review before merging
- Used GitHub Issues for task tracking

---

## Collaboration
This project simulates collaboration between two contributors using separate branches to demonstrate real-world GitHub workflows.
